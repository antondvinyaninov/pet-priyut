# Вспомогательные скрипты

Эта папка содержит вспомогательные скрипты для разработки и обслуживания проекта.

## Структура

### tests/
Тестовые скрипты для проверки функционала импорта и парсинга карточек животных.

### analysis/
Скрипты для анализа данных, поиска дубликатов и пропущенных карточек.

### import_helpers/
Скрипты для импорта данных из документов Word в базу данных.

## Использование

Эти скрипты предназначены для разовых задач и отладки. Они не являются частью основного приложения.

Для запуска скриптов используйте:
```bash
php scripts/tests/test_card_8.php
php scripts/analysis/find_all_missing.php
bash scripts/import_helpers/import_all_missing.sh
```
